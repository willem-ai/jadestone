=== 翡翠原石皮壳→内部推断规则 ===

【砂粒与种水】
- 砂粒极细密(如面粉,肉眼无法分辨单粒) → 内部大概率冰种及以上, 置信度0.7
- 砂粒细密(可见细微颗粒但均匀) → 糯冰至冰种, 置信度0.6
- 砂粒中等(可辨颗粒,触感有砂感) → 糯种至糯冰, 置信度0.7
- 砂粒粗松(颗粒明显,排列松散) → 豆种至糯种, 置信度0.8 (最可靠负向指标)
- 砂粒粗细不均 → 内部结构可能不均匀, 变种风险高

【打灯与种水】
- 透光>3cm且光圈大而均匀 → 冰种及以上, 置信度0.75
- 透光1-2cm,光圈中等 → 糯冰至冰种, 置信度0.6
- 透光<1cm,光圈小 → 豆种至糯种, 置信度0.8
- 透光不均匀(斑块状) → 内部结构不一致, 可能有棉团或变种
- 透光颜色偏绿 → 内部可能有绿色底或飘绿
- 透光颜色偏白 → 无色或白色底
- 透光颜色偏黄 → 黄翡底
- 透光颜色偏紫 → 紫罗兰底

【松花与绿色】
- 松花带状连续+色深 → 内部有绿概率0.7, 绿色浓度中高
- 松花团块状+色深 → 内部团状绿概率0.6
- 松花点状分散+色浅 → 内部淡绿或无色概率0.7
- 松花仅零星 → 内部大概率无色
- "绿随松花走"是行业共识, 但松花不进入内部(仅表皮绿)的情况约占30%

【蟒带与色带】
- 宽蟒(>2cm)+边界清晰 → 蟒下有色带概率0.75
- 中等蟒(1-2cm)+较清晰 → 可能有色带, 置信度0.55
- 窄蟒(<1cm)+模糊 → 可能仅有浅色带或无色, 置信度0.5
- 蟒带凸起 → 内部结构可能致密
- 蟒带环绕 → 色带可能环绕内部

【癣的判定】
- 黑癣在绿松花旁边 → "癣夹绿", 是好信号, 置信度0.6
- 黑癣在松花正上方 → "癣吃色", 是坏信号, 内部可能被癣破坏
- 癣独立无松花相伴 → 纯杂质, 负面影响
- 灰癣/猪鬃癣 → 影响相对较小

【裂纹评估】
- 细如发丝(hairline), 表面裂纹 → 深入内部概率约30%
- 肉眼可见开口, 宽度>0.5mm → 深入内部概率约60-70%
- 多条平行裂纹 → 内部结构可能疏松, 风险高
- 裂纹在正面中心 → 破坏手镯位, 价值影响大
- 裂纹在边缘 → 影响相对小
- 打灯观察裂纹处: 如果透光在裂纹处中断→裂纹已深入

【皮壳类型与场口先验】
- 木那白沙皮: 冰种率较高, 常出飘花/晴水
- 莫西沙黑乌沙: 冰糯种居多, 常出飘绿
- 莫西沙白沙皮: 冰种率较高, 常出无色冰种
- 后江水翻砂: 糯种居多, 但颜色常好
- 大马坎黄沙皮: 糯冰居多, 常出黄加绿
- 会卡青蛙皮: 种水较好, 但裂多
- 帕敢黑乌沙: 赌性大, 好料率低但一旦好就是极品

【处理货(B/C货)鉴别】
- 酸洗纹: 网格状/蜘蛛网状纹理 → B货风险高
- 染色: 颜色沿裂纹富集, 色根不自然 → C货风险高
- 注胶: 表面异常胶状光泽, 紫外荧光异常 → B货风险高
- 皮壳人工做旧: 皮壳颜色过于均匀, 无自然氧化过渡 → 造假风险
